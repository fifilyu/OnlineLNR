说明

本系统为开源项目，代码地址： OnlineLNR。 请遵循GPL3.0许可使用本代码。如有任何问题或建议，请提交 New issues 。

文件格式

你可以上传一个小于2M的图片，格式支持 JPG、JPEG、PNG。 可以使用拖拽或选择文件的方式上传包含车牌号的图片。

浏览器支持

经过编码的结果与以下所有浏览器和版本兼容。如果有什么问题，请给我联系我。

  • Chrome 4+
  • Safari 3.1+
  • Firefox 2+
  • Opera 9+
  • Edge
  • IE 8+
